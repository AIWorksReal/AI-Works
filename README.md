# AI Works one-page website

Gatava vienas lapas mājaslapa.

## Kā lietot
1. Atver mapi VS Code.
2. Atver `index.html` pārlūkā vai ar Live Server.
3. Maini tekstus `index.html` failā.
4. Dizains atrodas `css/style.css`.
5. Animācijas un mobilā izvēlne atrodas `js/script.js`.

Kontaktforma izmanto `mailto:worksai@inbox.lv`, tāpēc tā atver lietotāja e-pasta programmu. Ja vēlāk vajadzēs īstu servera formu, var pieslēgt Formspree, Netlify Forms vai servera risinājumu.
